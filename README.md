# CBT Therapy Bot for Deta Space

این پروژه یک بات تلگرامه که با استفاده از GPT پاسخ‌های درمان شناختی رفتاری (CBT) ارائه می‌ده.

## دستورات

- `/start` — شروع بات  
- `/ask <سوال>` — ارسال سوال به GPT

## راه‌اندازی

1. توکن‌های `TG_TOKEN` و `OPENAI_API_KEY` رو در محیط Deta ثبت کن  
2. این فایل‌ها رو در پروژه‌ی Deta قرار بده  
3. پروژه رو اجرا کن و در تلگرام تستش کن
